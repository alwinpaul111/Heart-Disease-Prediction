# Heart-Disease-Prediction
This project focuses on predicting the risk of heart disease in patients using clinical health indicators and machine learning algorithms.
The goal is to build a predictive model that can help identify patients who are at higher risk, enabling earlier medical intervention and better healthcare decision-making.

The project implements Logistic Regression and Random Forest models, evaluates performance using Accuracy, F1-score, and ROC-AUC, and performs exploratory data analysis and feature importance analysis.
